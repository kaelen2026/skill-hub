Features

Read the truth.
Then change it, safely.

Read-only overview

Scan all scopes, parse frontmatter, aggregate by name. Drift, broken, shared, duplicated — surfaced at a glance before you touch a thing.

PHASE 1

Enable, disable, relocate

Toggle a skill by flipping SKILL.md ↔ .disabled. Promote to the shared store with an automatic symlink back. Link and unlink per project.

PHASE 2

Built-in SKILL.md editor

Lossless raw-text editing with live Rust-side validation — required fields, YAML, format detection. Refuses to touch Codex .system bundles.

PHASE 3

Cross-tool sync

Project frontmatter Claude ↔ Codex with the body as canonical source. Generates agents/openai.yaml, preserves when_to_use into metadata — no fabricated icons.

PHASE 4

Preview, then back up

Every write shows a unified diff first, then auto-backs-up to ~/.skill-hub/backups/<ts>/ before a single byte changes on disk. Nothing is irreversible.

SAFE WRITES

Native, offline, fast

Built on Tauri — a Rust core with a React/TS UI. No Electron bloat, no cloud round-trips. Your skills never leave your machine.

Quick Start

Running in
three steps.

Requires Rust (via rustup) and Node. macOS 12+.

1

Clone & install

Pull the repo and install frontend deps with npm install. Tauri pulls the Rust toolchain on first build.

2

Run in dev

Launch the desktop app with npm run tauri dev. It scans your scopes read-only on first open — nothing is written.

3

Review & sync

Spot drift, enable what you need, edit a SKILL.md, or sync Claude ↔ Codex. Every change previews a diff and backs up first.

terminal
# 1 — clone & install git clone https://github.com/you/skill-hub cd skill-hub && npm install # 2 — run the desktop app npm run tauri dev # 3 — build a release .app npm run tauri build

⚠ cargo not on PATH?

rustup installed via Homebrew has no ~/.cargo/bin shim, so tauri dev can't find cargo. Prefix the command:

PATH="$HOME/.rustup/toolchains/stable-aarch64-apple-darwin/bin:$PATH" npm run tauri dev
